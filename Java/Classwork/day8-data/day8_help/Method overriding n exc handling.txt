Method overriding n exception handling
Help : Overriding form of the method(in sub class) can't add any NEW or BROADER checked exceptions
Confirm with examples.
eg :
1.  package p1;
class A
{
  void show()
  {
    sop("1");
  }
}

class B extends A
{
  @Override
  void show() throws InterruptedException 
  {
    sop("2");
  }
}

In Tester 
A ref=new B();
ref.show();
